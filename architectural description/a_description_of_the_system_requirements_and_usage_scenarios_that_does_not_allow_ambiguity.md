#Game Launch Instructions

1. To start the game, enter in terminal:
make main
./main

Copy

2. Game flow:
   - Choose whether to play (1/0)
   - If playing (1):
     - Select game mode
     - Choose who moves first
   - Game begins

3. After game ends:
   - Choose to play again (1/0)
   - If "0" selected, game exits
